Program ini digunakan untuk menghitung volume tabung berdasarkan jari jari jari jari tinggi yang sudah ditentukan.
Program punya fungsi bernama hitungVolumeKubus yang menerima tiga angka: panjang, lebar, dan tinggi.
Fungsi tersebut menghitung volume dengan rumus:\text{volume} = \text{panjang} \times \text{lebar} \times \text{tinggi}
 Di bagian main, program mengisi nilai panjang, lebar, dan tinggi.
 Program memanggil fungsi untuk menghitung volume.
 Hasilnya ditampilkan ke layar.
